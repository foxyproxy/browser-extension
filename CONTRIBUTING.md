By submitting a contribution, you represent that:

1. You have the right to submit the contribution.
2. The contribution is your original work, or you have permission to submit it.
3. Your contribution does not knowingly violate the rights of any third party.
4. You understand that your contribution may be modified, redistributed, and incorporated into this project under the GNU General Public License version 2.0.

## How to Contribute

Contributions may be submitted through issues or github pull requests.

Before submitting a pull request, please:

1. Make sure your changes are relevant to the project.
2. Keep changes focused and reasonably scoped.
3. Include tests or documentation where appropriate.
4. Follow the existing code style and project structure.
5. Clearly describe the purpose of your change.

## Pull Requests

When submitting a pull request, please include:

- A clear description of the change.
- Any relevant github issue numbers.
- Notes about testing, compatibility, or migration concerns, if applicable.


## Code of Conduct

Contributors are expected to participate respectfully and constructively.

Harassment, abusive behavior, personal attacks, or intentionally disruptive conduct may result in removal from project discussions or rejection of contributions.

## Maintainer Discretion

The maintainers reserve the right to accept, reject, modify, or remove contributions at their discretion.

Submitting a contribution does not guarantee that it will be merged, included, or retained in the project.

## Questions

If you have questions about contributing, please open an issue or contact the project maintainers through the appropriate project channel.
